Installation guide

Install your Personal AI Workspace.

You need ChatGPT, one Notion account, the Notion connector, and the latest installer file. Everything else is optional and can be enabled later.

Before you start

Only Notion is essential.

Personal AI Workspace stores its durable structure and operating rules in Notion. Without a connected Notion account, there is no persistent workspace.

All other services are optional. You can begin with the core system and add email, files, calendars, task backends, professional tools or specialist plugins later.

Absolutely required

Core installation requirements

  • A ChatGPT accountIt must support file uploads and the Notion connector on the surface you are using.
  • A Notion accountUse the account and workspace where you want the system to live.
  • The Notion connectorConnect it before installation and verify the correct account identity.
  • The latest installer fileDownload the Markdown creator from the latest GitHub release.
  • A new ChatGPT conversationDo not install inside an old, unrelated thread with a large existing context.

Step by step

Complete the installation from beginning to end.

  1. 1

    Connect Notion to ChatGPT

    Open ChatGPT settings, find Apps or Connectors, choose Notion, and connect the account where the workspace should be created. Interface labels may vary slightly by product surface.

    Important: make sure the connected account is the intended owner account before allowing any write.

  2. 2

    Download the latest installer

    Open the latest release and download the file whose name starts with Personal-AI-Workspace-Creator- and ends with .md.

    Download the installer →
  3. 3

    Open a completely new ChatGPT conversation

    Use a fresh thread. Attach the downloaded Markdown file before sending the first instruction.

  4. 4

    Paste the installation prompt

    Follow the instructions in the attached file and guide me through the complete configuration from beginning to end. Do not summarize the file—start with the first stage.
  5. 5

    Answer the installer’s questions

    Choose the workspace owner identity, assistant name, optional integrations, task backend, file archive preference and other modules. You do not have to enable everything.

  6. 6

    Approve the exact workspace structure

    The installer should show the proposed structure, risks and rollback before structural writes. Review it, then approve only the scope you actually want installed.

  7. 7

    Run the final verification

    Confirm that the Constitution, Start Here, modules, registries and core databases can be read back completely. The installer should then give you the exact personalization text and optional project instruction to paste into ChatGPT.

Common problems

Stop when verification is incomplete.

The wrong Notion account is connected

Do not continue. Reconnect the correct account and restart the account verification step.

The Constitution is truncated

Treat it as not loaded. The installer must shorten or modularize it and then perform a complete readback.

An optional connector is unavailable

Continue with an honest unavailable or pending state. Optional integrations must not block the core Notion installation.

A file upload cannot be verified

Report the archive as incomplete. Never claim success from an upload request alone.
