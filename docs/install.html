Connect Notion before installation

First make sure you have a ChatGPT or Codex account and a Notion account. Then install or enable the Notion plugin/app in ChatGPT or Codex and connect it to the exact Notion workspace where you want Personal AI Workspace to live. Only then should you start the installation.

Automatic installation requires Notion create/edit capability. A read-only file-search connection is not enough. The wizard will show you how to verify this without changing anything.

Common blockers

Stop when a required check fails.

The wrong Notion account is connected

Disconnect it, connect the intended workspace, and rerun the identity check. Never install into the wrong account “just for now.”

Notion is read-only

Automatic installation cannot continue. Use a supported Notion plugin/app with create and update permissions, or wait for a documented manual fallback.

The Constitution is truncated

Treat it as not loaded. The installer must repair the layout and complete a full readback before meaningful work continues.

An optional integration is missing

Continue with the core system. Optional capabilities should be marked unavailable or pending, not allowed to block Notion installation.
