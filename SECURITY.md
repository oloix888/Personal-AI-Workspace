# Security policy

## Reporting a vulnerability

Please do not publish credentials, private connector data, personal information or exploitable security details in a public issue.

Report security-sensitive concerns to **michal24749@gmail.com** with the subject `Personal AI Workspace security report`. Include a concise description, affected version, reproduction steps and impact.

## Scope

This repository contains documentation and configuration workflows. Risks may include accidental disclosure, unsafe automation, consent bypass, connector misuse or preservation of secrets.
