# Public repository design

## Goal

Publish a professional English-language home for Personal AI Workspace with a clear brand, current and historical downloads, open-source attribution, community pathways and automated validation/releases.

## Architecture

- GitHub repository README for immediate orientation
- Static GitHub Pages site under `/docs`
- Creator files under `/dist` and `/archive`
- Release metadata in `release-index.json`
- Actions for validation, Pages and GitHub Releases
- Apache-2.0 with NOTICE, AUTHORS and attribution guidance

## Visual system

Burnt orange and amber gradients, warm white surfaces, high-contrast white hero text only over dark orange, restrained network imagery and system typography.
