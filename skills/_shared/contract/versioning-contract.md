# Versioning contract

Framework, Skill Pack, individual skill, and installed Workspace schema versions are tracked independently. Unknown newer Workspace versions are read-only and are never automatically downgraded.

The public Skill Pack version is `0.1.0-beta.1`. The compatibility manifest records the supported framework versions and the Phase 1 framework / Markdown creator target, `1.6.0`. Markdown creator `1.5.1` remains the supported universal fallback.
